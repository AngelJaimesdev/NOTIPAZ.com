¿ Funcion de cada miembro del proyecto ?

En la institución universitaria de la paz (UNIPAZ), un grupo de estudiantes vieron la necesidad de satisfacer las necesidades detectadas en el manejo y tratamiento de la información academica que periodicamente se realiza en el instituto universitario de la paz (UNIPAZ), con el fin de garantizar y agilizar los procesos.

Para eso tres estudiantes han tomado la iniciativa para comenzar este proyecto dividiendose las tareas :

1). Documentación, entrevistas y pagina web: El estudiante Angel Manuel Jaimes Vargas es el encargado de estas funciones, la pagina de iniciar sesión para los estudiantes, tanto la documentación del proceso y la información obtenida gracias a las entrevistas a diferentes docentes de escuela de ingenieria de producción.

2).Implementación de la base de datos: El estudiante luis Enrique Suarez ya conocido en la institución por sus aportes y participaciones en otros proyectos, tiene la función de crear la base de datos para ellos pidio permiso a la escuela de ingenieria de producción para poder acceder a la cantidad de estudiantes que hay actualmente y de asi crear su propia base de datos con la información de la institución.

3).implementación del modelo del software: El estudiante Jesus Raul Villalobos tiene la funcion de crear el modelo del software usando programas conocidos como java Todas las funciones de los estudiantes se conectaran entre si para tener el proyecto completo, para ello en la pestaña que dice NOTIPAZ hay mas información.
